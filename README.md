# M8L1

### Моя идея проекта: «Потоп» (Flood)
Вертикальный скроллер-выживание, где игрок управляет дроном, собирающим технологии спасения, пока уровень воды неумолимо поднимается.

### Тип моего проекта:
Десктопная 2D-игра на Python с элементами симуляции выживания и процедурной генерации. Игра создаёт атмосферу надвигающейся катастрофы через механику постоянно поднимающегося уровня воды и ограниченные ресурсы. Игрок управляет дроном (кликами мыши или клавишами), который должен собирать артефакты (солнечные панели, семена, чертежи) с крыш затопленных зданий, уклоняясь от молний, обломков и штормового ветра.

### Библиотеки, которые я буду использовать

- NumPy

Создание процедурно генерируемого фона (здания, волны) с помощью шума Перлина или случайных чисел [Procedural Generation in Pygame](https://www.youtube.com/watch?v=procedural_generation_pygame)

Реализация системы здоровья дрона и получения урона от препятствий [Pygame Health/Damage System](https://www.youtube.com/watch?v=pygame_health)
